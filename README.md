# Representative Learning (RepL)

This repository presents research on **Representative Learning (RepL)**, a data-centric framework for distributed machine learning and distributed AI.

Unlike traditional approaches that rely on model or gradient sharing, RepL enables nodes to communicate **structured pseudo-data ("representatives")** that preserve essential statistical information while respecting communication, privacy, and system constraints.

## Key Themes
- Distributed learning under heterogeneous, non-IID data
- Communication-efficient and privacy-preserving algorithms
- Structured data representations instead of model sharing
- Asynchronous and decentralized systems

## Methodological Development
- Point representatives: MR, SMR, RASMR, TMR, Anchored-SMR
- Extensions to smooth ERM and generalized linear models
- Non-smooth learning: SVM, quantile regression, clouded representatives
- Functional data analysis and distributed optimization

## Goal
To establish a unified, interpretable, and scalable framework for **distributed AI through data-centric communication**.
